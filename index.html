  • Videogame Catalog

    The fully responsive application dayZgamer, a videogame catalog, was designed with the technologies above best practises, utilizing the RAWG API.   • Nutriton Tracker

    HTML5, CSS3 [BEM], SASS, JavaScript ES6+, React [Hooks], Git, REST API

    The fully responsive application dayZtracker, a nutrition tracker, was designed with the technologies above best practises, utilizing the nutritonix API. The landing page provides some helpful information about how the application can help you reach your goal. After inputting a calorie goal the user is redirected to another page that tracks everything on the nutrition label. The user inputs their meal and the application will track everything, including their progress on their goal.

  • Book E-Commerce Website

    HTML5, CSS3 [BEM], JavaScript ES6+, React [Hooks + Dev Tools], Git

    The fully responsive application Library, a book e-commerce website, was designed with the technologies above best practises. The landing page provides an assortment of featured books as well as their price and rating. The cart has robust logic and allows the user to add and remove items. The price will update as needed incorpoarting tax. If the user wants more information about the book, they can click on it to read more.
